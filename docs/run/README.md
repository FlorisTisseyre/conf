## Variante 90 min (créneau serré)

On rogne les deux phases (~25 min chacune) et les collectes de débrief.
**On ne coupe jamais le quiz ni le re-ressenti** : c'est le pivot.
Premier truc à sacrifier si ça déborde : la collecte du débrief phase 2 (2–3
binômes au lieu de tous), **pas** le tour de clôture.

## Règle d'or (à répéter 3×)

On ne devient ni augmenté ni aliéné en 30 min. L'atelier fait **sentir une
direction**. Personne n'est étiqueté. Ce n'est pas un examen.
